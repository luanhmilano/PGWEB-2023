# Programação WEB

Repositório dedicado à disciplina de **Programação WEB** do curso de Análise e Desenvolvimento de Sistemas do IFPA - Campus Belém, realizada no 3° Semestre (2023.2). Aqui serão armazenados códigos-exemplo, exercícios e projetos desenvolvidos durante a disciplina.

![Status](https://img.shields.io/badge/Status-Finalizado-blue)

## Sobre a Disciplina
Disciplina fundamental para o desenvolvimento web moderno, abordando ponto a ponto algumas das tecnologias de desenvolvimento client-side e server-side.

**Competências Desenvolvidas:**
- Desenvolvimento Frontend.
- Desenvolvimento Backend.
- Entendimento de protocolos e requisições.
- Conhecimento de ferramentas e frameworks web.

## Ementa da Disciplina
- Fundamentos sobre Programação para web.
- Arquitetura da Internet e Web (WWW).
- Programação distribuída no modelo C/S.
- Programação na Camada Cliente (web).
- Linguagens de programação JS, Protocolo HTTP, HTML 5, CSS.
- Frameworks de apoio à programação, ferramentas de programação.
- Programação na Camada servidor.
- Integração com banco de dados e WebServices.

## Stack Tecnológica
**Ferramentas:** 
- HTML
- CSS
- JavaScript
- Bootstrap
